ภาษา HTML

HTML (HyperText Markup Language) เป็นภาษาคอมพิวเตอร์ที่ใช้ในการสร้างโครงสร้างและเนื้อหาของหน้าเว็บไซต์ โดย HTML จะใช้ตัวอักษรและสัญลักษณ์พิเศษเพื่อให้เข้าใจได้ว่าเนื้อหาบนหน้าเว็บไซต์นั้นมีลักษณะอย่างไร และวางโครงสร้างเนื้อหานั้นอยู่ในลักษณะที่ถูกต้องตามมาตรฐานของเว็บไซต์

หน้าเว็บไซต์ที่ถูกสร้างด้วย HTML จะประกอบด้วยหลายๆ องค์ประกอบ เช่น ข้อความ รูปภาพ ลิงก์ ตาราง ฟอร์ม และอื่นๆ โดย HTML จะใช้ tag ในการกำหนดประเภทขององค์ประกอบและลักษณะของเนื้อหานั้น เช่น < p > สำหรับกำหนดข้อความเป็นย่อหน้า หรือ < img > สำหรับแทรกรูปภาพลงในหน้าเว็บไซต์ นอกจากนี้ HTML ยังสามารถใช้กับภาษาอื่นๆ เพื่อสร้างเนื้อหาที่มีลักษณะพิเศษ เช่น JavaScript หรือ CSS ในการสร้างเนื้อหาที่เป็นแบบพิเศษ หรือ PHP ในการจัดการข้อมูลในฐานข้อมูลของเว็บไซต์นั้นๆ

HTML5 คือเวอร์ชันล่าสุดของภาษา HTML โดย HTML5 มีการพัฒนาเพื่อตอบสนองความต้องการของการใช้งานเว็บไซต์ในยุคปัจจุบัน โดยมีคุณสมบัติและฟีเจอร์ที่มากกว่าเวอร์ชันก่อนหน้า ดังนี้:

  • รองรับการแสดงผลสำหรับอุปกรณ์ที่แตกต่างกัน และมีการปรับขนาดหน้าจอแบบอัตโนมัติ (Responsive Design)
  • มีการเพิ่มฟอร์มที่มีความสามารถในการตรวจสอบข้อมูลของผู้ใช้งานก่อนส่งข้อมูล (Form validation)
  • มีการเพิ่มฟีเจอร์ในการเล่นเสียงและวิดีโอโดยไม่ต้องใช้โปรแกรมเสริม เช่น การใช้งานแท็ก video, audio
  • มีการเพิ่มฟีเจอร์ในการทำงานแบบออฟไลน์ (Offline Web Application)
  • มีการเพิ่มฟีเจอร์ในการสร้างแอปพลิเคชันบนเว็บไซต์ (Web Application)
  • มีการเพิ่มฟีเจอร์ในการเข้าถึงฮาร์ดแวร์บนอุปกรณ์ของผู้ใช้งาน เช่น กล้องถ่ายภาพ ไมค์ และเซ็นเซอร์ต่างๆ

ภาษา CSS

CSS (Cascading Style Sheets) เป็นภาษาที่ใช้สำหรับออกแบบหน้าเว็บไซต์ โดยมีหน้าที่กำหนดรูปแบบการแสดงผลของ HTML (HyperText Markup Language) โดยเฉพาะเลยที่สุด เช่น สีพื้นหลัง (background color), รูปแบบตัวอักษร (font style), ขนาดตัวอักษร (font size), การจัดวาง (layout), การเลือกแสดงข้อมูล (visibility), และอื่น ๆ อีกมากมาย

CSS มีการทำงานร่วมกับ HTML เพื่อจัดการแสดงผลหน้าเว็บไซต์ ด้วยการกำหนดคุณสมบัติต่าง ๆ ให้กับตัวองค์ประกอบต่าง ๆ ใน HTML ดังนั้น CSS จะช่วยให้การออกแบบหน้าเว็บไซต์เป็นไปอย่างเหมาะสมและสวยงาม

CSS3 (Cascading Style Sheets Level 3) เป็นเวอร์ชันล่าสุดของภาษา CSS ซึ่งมีการพัฒนาขึ้นเพื่อเพิ่มความสามารถและความหลากหลายในการออกแบบหน้าเว็บไซต์ โดยมีฟีเจอร์ต่าง ๆ ที่ให้ผู้ออกแบบสามารถสร้างและปรับแต่งสไตล์การแสดงผลได้อย่างคล่องตัวและมีความยืดหยุ่นมากขึ้น CSS3 มีฟีเจอร์ที่สำคัญ ๆ ดังนี้:

  • Selectors: CSS3 มีเล็กเล็ตเลือกตัวเลือกและการแสดงผลในโค้ด CSS ที่ยืดหยุ่นมากขึ้น รวมถึงเลือกตัวอย่างเป็นพิเศษ เช่น nth-child() ,nth-of-type() และแบบเลือกตัวเลือกแบบอื่น ๆ
  • Box Model: CSS3 มีคุณสมบัติใหม่ในกล่องโมเดลที่ช่วยเพิ่มความยืดหยุ่นในการจัดวางและสร้างสไตล์ด้วยช่องว่างระหว่างเนื้อหา (padding) และเส้นขอบ (border) ที่ต่างกัน
  • Backgrounds and Borders: CSS3 มีคุณสมบัติการตั้งค่าภาพพื้นหลังเป็นสไตล์เต็มรูปแบบ รวมถึงการเปลี่ยนแปลงต่อเนื่องสำหรับสีพื้นหลังและเส้นขอบ
  • Text Effects: CSS3 มีคุณสมบัติการเพิ่มเอฟเฟ็กต์การแสดงผลในข้อความ เช่น การปรับแต่งตัวอักษรด้วยการเปลี่ยนสีเมื่อเม้าส์เลื่อนไปบนมัน
  • 2D/3D Transformations: CSS3 มีคุณสมบัติ

ภาษา JAVASCRIPT

Javascript เป็นภาษาโปรแกรมมิ่งที่ใช้สำหรับเว็บไซต์และแอปพลิเคชันเว็บ มีหน้าที่ในการเพิ่มความสามารถและประสิทธิภาพให้กับเว็บไซต์หรือแอปพลิเคชันเว็บ โดยการใช้ Javascript ผู้พัฒนาสามารถเพิ่มการตอบสนองแบบ real-time ให้กับผู้ใช้งานได้ สร้างการทำงานแบบฝังตัวลึก (deep integration) กับผู้ใช้งาน และให้ประสิทธิภาพการทำงานที่รวดเร็วและสามารถปรับเปลี่ยนได้ง่าย

โดย Javascript มีลักษณะเป็นภาษาโปรแกรมมิ่งที่แบบฝังตัว (embedded) ที่ใช้งานได้บนเว็บเบราว์เซอร์และแอปพลิเคชันเว็บ โดยใช้ภาษา HTML และ CSS เป็นฐาน จึงทำให้การใช้งาน Javascript ในการพัฒนาเว็บไซต์หรือแอปพลิเคชันเว็บมีความสะดวกและอำนวยความสะดวกสูง นอกจากนี้ Javascript ยังมีความสามารถในการเชื่อมต่อกับ API ต่าง ๆ และสร้างการทำงานแบบ asynchronous ที่ช่วยเพิ่มประสิทธิภาพการทำงานของเว็บไซต์หรือแอปพลิเคชันเว็บได้อีกด้วย

มีฟีเจอร์ของภาษา Javascript ที่สำคัญ ๆ ดังนี้:

  • ฟังก์ชัน (Function) - ฟีเจอร์สำคัญในการสร้างโปรแกรมในภาษา Javascript โดยมีการเรียกใช้งานและส่งค่าพารามิเตอร์ได้
  • อาร์เรย์ (Array) - โครงสร้างข้อมูลสำหรับเก็บข้อมูลหลายๆ ชิ้น ซึ่งสามารถดึงข้อมูลแต่ละตัวได้โดยใช้ index
  • ออบเจ็กต์ (Object) - โครงสร้างข้อมูลสำหรับเก็บข้อมูลแบบ key-value pairs ซึ่งใช้ในการแทนแอปพลิเคชันหลายๆ ประเภท เช่น JSON
  • ฟังก์ชัน callback - ฟีเจอร์ที่มีการใช้งานอย่างแพร่หลายในภาษา Javascript โดยใช้ function ในการเรียกใช้งานตัวอื่น ๆ แบบ asynchronous
  • Promise - ฟีเจอร์ในการจัดการกับการทำงานแบบ asynchronous ซึ่งช่วยให้ง่ายต่อการจัดการกับข้อมูลในแต่ละสถานะที่เปลี่ยนแปลงได้
  • Async/await - ฟีเจอร์ในการจัดการกับการทำงานแบบ asynchronous ที่ทำให้เขียนโค้ดที่เป็น Promise ได้ง่ายขึ้น
  • Template literals - ฟีเจอร์ในการสร้างสตริง (string) ที่ช่วยให้เขียนโค้ดได้อย่างสะดวกและคล้ายกับการเขียน HTML หรือ XML
  • Spread syntax - ฟีเจอร์ในการใช้ spread operator (...) ในการแยกสมาชิกของอาร์เรย์ หรือ object ออกเป็นตัวแปรเดียวกัน
  • Destructuring assignment - ฟีเจอร์ในการกำหนดค่าของ object หรือ array ให้กับตัวแปร

ภาษา PHP

PHP คือภาษาโปรแกรมมิ่งซึ่งถูกพัฒนาขึ้นโดย Rasmus Lerdorf ในปี 1994 และเป็นภาษาที่มีประสิทธิภาพสูงในการสร้างและดำเนินเว็บไซต์และแอปพลิเคชันเว็บโดยเฉพาะ เป็นภาษาที่ใช้งานกันอย่างแพร่หลายในการพัฒนาเว็บไซต์เนื่องจากมีการสร้างเว็บไซต์ได้ง่ายและมีความยืดหยุ่นในการใช้งาน

ภาษา PHP เป็นภาษาที่ใช้งานบนเว็บเซิร์ฟเวอร์ ซึ่งสามารถประมวลผลข้อมูลและส่งผลลัพธ์กลับไปยังเว็บบราวเซอร์ได้ ภาษา PHP สามารถใช้สร้างหน้าเว็บไซต์แบบไดนามิก และสามารถเชื่อมต่อกับฐานข้อมูลเพื่อดึงข้อมูลและแสดงผลได้

PHP มีฟังก์ชันที่สำคัญ ๆ ดังนี้:

  • String functions - ใช้สำหรับจัดการกับข้อมูลประเภท string เช่น strlen() , substr() , explode() ฯลฯ
  • Array functions - ใช้สำหรับจัดการกับข้อมูลประเภท array เช่น count() , sort() , array_key_exists() ฯลฯ
  • File functions - ใช้สำหรับจัดการกับไฟล์ เช่น fopen() , fclose() , fread() , fwrite() ฯลฯ
  • Database functions - ใช้สำหรับเชื่อมต่อและจัดการกับฐานข้อมูล เช่น mysqli_connect() , mysqli_query() , mysqli_fetch_assoc() ฯลฯ
  • Date and time functions - ใช้สำหรับจัดการกับวันที่และเวลา เช่น date() , time() , strtotime() ฯลฯ
  • Math functions - ใช้สำหรับการคำนวณเลขคณิต เช่น rand() , floor() , ceil
